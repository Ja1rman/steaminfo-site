SteamInfo.ru - Онлайн инструмент для поиска и конвертирования steam id в популярные форматы. community ID (custom url) / steamID / steamID32 (AccountID) / steamID64 / URL / steamID3 (USteamID).

Что такое steam id?
SteamID - это уникальный идентификатор, используемый для определения Стим аккаунта. Он также используется для ссылки на страницу профиля пользователя в сообществе Steam.
